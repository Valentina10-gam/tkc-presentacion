Cómo iniciamos

Llegamos a la jornada con una base lista para decidir y producir.

La idea es presentar una base inicial de contenidos para redes: qué servicios priorizar, qué empezar a publicar y cómo conectar esos contenidos con videos, web y pauta.

En la reunión
  • Mostrar propuestas de temas para redes.
  • Validar servicios y públicos prioritarios.
  • Definir qué videos de procedimientos grabar primero.
  • Aprobar el enfoque visual y comercial inicial.
Después de aprobar

Se arma el calendario del primer mes, se organizan las grabaciones y se inicia la producción de piezas para activar redes completamente.
