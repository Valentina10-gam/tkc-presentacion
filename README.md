# Presentación TKC

Presentación HTML interactiva para la propuesta de marketing, web e identidad corporativa de TKC Fumigaciones. Está preparada para publicarse como sitio estático en GitHub Pages.

## Estructura

- `index.html`: presentación principal.
- `assets/logo-mark.png`: logo de TKC usado en la presentación.

## Cómo verla

Abre `index.html` en el navegador.

Controles:

- Flecha derecha, espacio o click: avanzar.
- Flecha izquierda o backspace: retroceder.
- `F`: pantalla completa.

## Publicar en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir todo el contenido de esta carpeta a la raíz del repositorio.
3. Entrar a `Settings > Pages`.
4. En `Build and deployment`, seleccionar `Deploy from a branch`.
5. Elegir la rama `main` y la carpeta `/root`.
6. Guardar y esperar el enlace de publicación.

## Nota

La presentación es estática: no requiere instalación, servidor ni dependencias externas.
